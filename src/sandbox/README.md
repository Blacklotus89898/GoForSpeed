# Practice Playground

## Commands

```bash
go version

# Initializing a workspace
go mod init go-for-speed

# Format
go fmt

# Build and run
go run main.go # executed in memeory

# or 
go build -o myapp main.go # produce executable
./myapp

# Test
go test

# Clean up
go mod tidy

# External Packages
go get github.com/google/uuid


```
